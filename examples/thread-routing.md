# Large task with thread routing

ユーザーはスレッド数やエージェント割り当てを書かず、次だけ入力します。

```text
@Auto Router スレッドを使って、このリポジトリ全体を領域別に監査してください。根拠のあるIssue候補だけを重要度順に統合し、変更はしないでください。
```

Auto Routerは自動的に次を行います。

- 親のSol/Terraが非重複の作業領域を定義する
- 判断が必要な領域はTerraスレッドへ割り当てる
- 明確で機械的な一覧化はLunaスレッドへ割り当てる
- Terraスレッドは担当領域内でLunaサブエージェントを使用できる
- Lunaスレッドはサブエージェントを使用しない
- 同時実行上限に達する場合は複数の波に分ける
- 親が結果を検証、重複排除、統合する
